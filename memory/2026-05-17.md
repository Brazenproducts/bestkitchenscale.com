## Amazon Affiliate Tag Fix — 2026-05-17 ~19:35 UTC

### The Bug
**52 HTML files across 18 affiliate sites had the wrong Amazon tracking tag.**
- Sites were using `tag=brazenprodu01-20-magnesium` (missing `-20` suffix)
- Amazon's registered tracking IDs are `brazenprodu01-20-magnesium-20`
- Mismatch meant clicks were registering but conversions were likely misattributed or lost

### Evidence
- May 15: 201 clicks, 0 conversions, $0 revenue — Mitch flagged this
- Overall 30-day: 564 clicks, 8 orders, 1.42% conversion, $4.58 earnings

### Root Cause
When sub-agents built the sites, they set the tag as `brazenprodu01-20-{category}` but the Amazon Associates tracking IDs are formatted as `brazenprodu01-20-{category}-20`. The `-20` suffix is part of every Amazon Associates tracking ID and was being dropped.

### Fix Applied
- Found all 52 affected files with `grep`
- `sed` replaced every wrong tag with the correct `-20` suffixed version
- Committed and pushed all 17 affected site repos to GitHub Pages
